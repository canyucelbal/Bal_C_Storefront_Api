### Bikestore Backend API

    Backend API source code for bikestore

### System Setup

    In order to start server, change content of the ```./config/.env``` file

    ```DB_USERNAME```
    ```DB_PASSWORD```
    ```DB_NAME```
    ```DB_HOST```

    ```DEFAULT_PORT```

    After setting fields above, run

    ```npm install```

    ```npm run dev```

    API will be listening on the port ```DEFAULT_PORT```
